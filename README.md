
# BuBilet Crawler

Bubilet.com.tr websitesindeki etkinlikleri il ve kategori bazlı olarak bir excel dosyası halinde sizlerle buluşturuyor. Gitmek istediğiniz etkinliklerin hepsini bir sayfada detaylı olarak göreceksiniz.
